Copy the following files into this directory (these are all in the Spring distribution):

- spring.jar
- commons-logging.jar
- servlet-api.jar


Also needed for the Internet Bookstore:

- jstl.jar         (available from http://java.sun.com/products/jsp/jstl/ )
- junit-3.8.1.jar  (see http://junit.org )
- standard.jar     (jakarta-taglibs, included with the Spring distribution)
- hsqldb.jar       (available from http://hsqldb.org/)
